# PiWedge pinout
Mappinge til PiWedge er ikke 1 til 1 med GPIO pinsene derfor trenger vi denne

PWM0 (GPIO6) -> G4
PWM1 (GPIO7) -> TX
PWM2 (GPIO9) -> RX

5V -> G17
GND(5V) -> G18

DI3 (GPIO33) -> G20 
DI2 (GPIO32) -> G26
DI1 (GPIO31) -> G16
DI0 (GPIO30) -> G19

CE (GPIO25) -> IDSC
SCK (GPIO24) -> IDSD

3V3 -> G5
GND (3V3) -> GND
